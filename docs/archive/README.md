## 归档规则

1. **触发条件**：需求/计划的所有任务标记为 Completed 后立即归档
2. **目录结构**：按 `YYYY-MM/` 存放
3. **严格只读**：归档后不允许修改。如需引用或更新，复制到活跃目录并标注原 archive 位置
4. **设计文档不归档**：架构设计在 `designs/` 下持续演进
